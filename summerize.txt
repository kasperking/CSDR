Tóm tắt session CSDR STM32H750 TX DSP v4
Ngữ cảnh
Dự án CSDR SDR Transceiver trên STM32H750VBT6 (128KB flash), kiến trúc BSP-only (main.c chỉ 3 dòng USER, logic tập trung trong BSP/Src/csdr_app.c).

Các vấn đề đã giải quyết
1. RX FFT bandscope full-band (đã fix từ v3 cuối, confirm session này)

FFT 256-point (từ 128-point half-band), feed từ mix_i/mix_q sau NCO trước FIR
Hiển thị full ±24kHz thay vì half-band
Mode-aware BW markers (LSB/USB/CW asymmetric, AM/FM symmetric)

2. USB Composite Device (CAT + Audio full-duplex) — usb_composite.c standalone

5 interfaces: AC/AS_OUT/AS_IN (IAD Audio) + CDC_CTRL/CDC_DATA (IAD CDC)
Endpoints: EP1 IN/OUT bulk (CDC), EP2 IN intr (CDC notif), EP3 IN/OUT iso (Audio)
Self-contained, bypass USBD_CDC lib để an toàn khi CubeMX regenerate
Fix cascade:

SOF callback phải ENABLE trong usbd_conf.c (CubeMX default = DISABLE → Audio IN pipeline không pump)
SOF-pump pattern cho Audio IN: kick-start trong SET_INTERFACE + retry trong Comp_SOF + Comp_IsoInInc
Ring buffer 8×192 → 32×192 bytes (tránh overrun khi SAI block 1024B > ring 1536B)
Duplicate CDC_* macro (conflict với usbd_cdc.h) → bỏ



3. CAT Protocol (Kenwood TS-2000)

PTT fix: "TX;" luôn set TX (không phải query), "RX;" release
Bỏ toàn bộ snprintf (12 calls) → custom helpers cat_u32/cat_str/cat_parse_u (tiết kiệm 5-15KB flash, bỏ stdio/printf library)
Bỏ strtoul → custom cat_parse_u (bỏ stdlib)

4. Flash optimization (vượt 128KB nhiều lần)

codec_test.c excluded từ build (user đã làm trước)
Stub diag.c + cal.c (giữ API, empty implementation) — tiết kiệm ~7-10KB
s_cfg_desc[] → const với wTotalLength hardcoded = 266
Bỏ debug counters đã xong việc (dbg_comp_*, dbg_cdc_*)
Hilbert TX dùng AM fallback cho FM/CW mode (bỏ cosf/sinf)
-Os optimization level

5. TX DSP Pipeline (full implementation)
Kiến trúc:
Audio source → Hilbert → I/Q modulate → FIR LPF → DC block → FFT display
                                                            → SAI TX DAC → QSE
Chi tiết:

Hilbert_t + TX_State_t thêm vào DSP_State_t
Hilbert_Init: odd-symmetric FIR, Hamming window, coefficient h[n] = 2/(π·(n-M)) × window
Hilbert_Process: FIR convolve với circular buffer
HILBERT_TAPS evolution: 31 → 15 (flash) → 15 (symmetric spectrum bug) → 31 (vẫn yếu) → 63 (SSB clean)

Simulation: @1kHz, 15-tap gain=0.31, 31-tap gain=0.61, 63-tap gain=0.94


DSP_ProcessTX modulation:

USB: I = audio_delayed, Q = +Hilbert(audio)
LSB: I = audio_delayed, Q = -Hilbert(audio)
AM: I = 0.5 + 0.5·audio, Q = 0
FM/CW: AM fallback


Internal 1kHz square-wave test tone khi USB ring trống (debug pipeline không cần PC)
DC block trên TX output (sau FIR, trước FFT) để loại "center carrier spike"
Reset DC block state trong DSP_SetMode tránh residue từ mode trước

6. CSDR_Loop routing

Ping/pong SAI callback route theo g_sdr.tx_mode:

TX: DSP_ProcessTX (ghi IQ vào SAI TX buffer)
RX: DSP_Process (decode SAI RX → audio)


cat_set_tx KHÔNG mute USB streaming khi TX (để PC audio đi vào tx_ring)
